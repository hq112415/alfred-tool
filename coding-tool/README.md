# Coding Tool

编程小工具箱 Alfred 插件，集成多种常用开发工具。

## ✨ 功能列表

### 📋 JSON 格式化
- **左右面板布局** — 左侧输入 JSON，右侧展示格式化结果
- **智能格式化** — 标准 JSON 格式化，支持自定义缩进（2 空格 / 4 空格 / Tab）
- **语法高亮** — 键、字符串、数字、布尔值、null 各有颜色区分
- **错误检测** — JSON 无效时尽力格式化，并红色高亮标出错误位置
- **删除空格** — 一键将 JSON 压缩为单行（Minify）
- **删除空格并转义** — 压缩后转义双引号，适合将 JSON 嵌入到字符串中
- **去除转义** — 将转义的 JSON 字符串还原为正常 JSON，自动识别并格式化
- **折叠展开** — 支持 JSON 节点的折叠 / 展开
- **搜索** — 全文搜索并高亮匹配结果，支持复制匹配的值或键值对
- **复制结果** — 快速复制格式化后的 JSON
- **拖拽导入** — 支持将 `.json` 文件拖拽到输入区域
- **粘贴自动格式化** — 粘贴 JSON 后自动格式化

### ⏱ 时间戳转换
- **当前时间戳** — 实时显示当前时间戳，支持秒 / 毫秒切换，一键复制
- **时间戳 → 日期** — 输入时间戳转为可读日期时间，自动识别秒 / 毫秒
- **日期 → 时间戳** — 输入日期时间转为时间戳，同时显示秒和毫秒结果

## 📦 安装

```bash
bash install.sh
```

安装后双击生成的 `CodingTool.alfredworkflow` 导入 Alfred。

## 🚀 使用方法

**方式一：快捷键（推荐）**

直接按 `⌘⇧T`（Cmd + Shift + T）打开编程小工具箱。

**方式二：关键字**

1. 唤起 Alfred（默认 `⌘ + Space`）
2. 输入 `ct`，回车即可打开

## ⌨️ 快捷键

### 全局

| 快捷键 | 功能 |
|--------|------|
| `⌘⇧T` | 打开 / 呼出 Coding Tool |
| `Tab` | 切换 JSON 格式化 / 时间戳转换 |
| `ESC` | 关闭窗口 |

### JSON 格式化

| 快捷键 | 功能 |
|--------|------|
| `⌘ + Enter` | 格式化 JSON |
| `⌘ + F` | 搜索 |
| `⌘⇧M` | 压缩 JSON |
| `⌘⇧C` | 复制格式化结果 |

### 时间戳转换

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 在输入框中按回车执行转换 |

## 🛠 技术说明

- Swift 原生应用，WKWebView 渲染界面
- 纯前端实现，无外部依赖
- Tab 页切换不同工具
- SIGUSR1 信号通信实现重复触发时切换窗口显示
