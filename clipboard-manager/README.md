# Clipboard Manager

一个强大的剪切板历史管理工具，支持历史记录、收藏和快速搜索。

## ✨ 功能特性

- **全局快捷键** — `⌘⌥V` 随时呼出 / 隐藏剪切板面板
- **剪切板历史** — 自动记录所有复制的文本内容（最多 1000 条）
- **收藏功能** — 重要内容可加星标收藏，永不丢失
- **快速搜索** — 实时搜索过滤剪切板内容
- **键盘驱动** — 上下选择、左右切 Tab、回车直接粘贴
- **智能去重** — 相同内容自动去重并更新时间
- **最近使用置顶** — 选中使用的条目自动置顶
- **失焦自动隐藏** — 点击其他窗口自动关闭面板
- **窗口可拖动** — 拖动标题栏区域移动面板位置

## 📦 安装

```bash
bash install.sh
```

安装后双击生成的 `ClipboardManager.alfredworkflow` 导入 Alfred。

## 🚀 使用方法

1. 在任意程序中按 `⌘⌥V` 呼出剪切板面板
2. 使用 `↑↓` 键选择条目
3. 按 `Enter` 将选中内容粘贴到当前光标位置

### ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘⌥V` | 呼出 / 隐藏面板 |
| `↑` / `↓` | 上下选择条目 |
| `←` / `→` | 切换标签页（历史 / 收藏） |
| `Tab` / `Shift+Tab` | 切换标签页 |
| `⌘F` | 聚焦搜索框 |
| `Enter` | 粘贴选中条目到当前应用 |
| `⌘S` | 收藏 / 取消收藏选中条目 |
| `⌘Delete` | 删除选中条目 |
| `Esc` | 隐藏面板 |

### 交互说明

- **搜索** — 呼出面板后直接输入即可搜索，搜索范围为当前标签页
- **收藏** — 选中条目后按 `⌘S`，或点击条目右侧的星标按钮
- **删除** — 选中条目后按 `⌘Delete`，或点击条目右侧的删除按钮
- **置顶** — 选中并使用（Enter）的条目会自动移到列表最前面
- **数据文件** — 点击面板底部「📂 数据文件」按钮可查看原始数据

## 💾 数据存储

- 数据目录：`~/.clipboard_manager/`
- 历史数据：`~/.clipboard_manager/data.json`
- 窗口配置：`~/.clipboard_manager/config.json`
- 默认保留最近 1000 条历史记录
- 收藏条目无数量限制

## ⚠️ 注意事项

1. **辅助功能权限**：首次使用时需在「系统设置 → 隐私与安全性 → 辅助功能」中授权 Alfred，否则无法模拟 `⌘V` 粘贴
2. **常驻后台**：首次 `⌘⌥V` 启动后程序会常驻后台持续监听剪切板变化，关闭窗口只是隐藏
3. **退出程序**：如需完全退出，可通过活动监视器结束 `clipboard_manager` 进程

## 🛠 技术说明

- Swift 原生应用，**纯 AppKit UI**（NSTableView + NSSearchField + NSTextView）
- 无 WebView 依赖，内存占用仅 ~10-15MB（对比旧版 WKWebView 方案 ~150MB）
- 0.5 秒轮询监听系统剪切板变化
- CGEvent 模拟键盘按键实现粘贴
- 数据以 JSON 格式存储
- 窗口浮动层级（floating），失焦自动隐藏
